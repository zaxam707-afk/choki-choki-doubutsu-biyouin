# チョキチョキ！どうぶつびよういん

幼児〜小学校低学年向けの、動物の毛をスワイプでカットして可愛くしてあげる体験型ゲームのデモです。

## 遊び方

1. スタートボタンをタップ
2. お客さんの動物をえらぶ（20種類）
3. 制限時間内に画面を指でなぞって毛をカット
4. 95%以上きれいにできれば早めにクリア！

## 公開URL

GitHub Pages で公開中：
**https://zaxam707-afk.github.io/choki-choki-doubutsu-biyouin/**

## 特徴

- 単一HTMLファイルで動く軽量デモ（Canvas + Web Audio + Vibration API）
- 失敗のない加点方式：なぞればなぞるほどきれいになる
- スマホ・タブレット・PCブラウザに対応（タッチ＆マウス両方）
- 削られている間、動物がくすぐったそうに反応

## ローカルで動かす

```
git clone https://github.com/zaxam707-afk/choki-choki-doubutsu-biyouin.git
cd choki-choki-doubutsu-biyouin
# index.html をブラウザで開くだけ
```
